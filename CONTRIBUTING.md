# Contributing to Flowbit AI Second Brain

Thanks for contributing to Flowbit Labs.

## What to contribute
- Better extraction for PDFs/web pages
- UI improvements
- Better chunking/search quality
- Documentation and examples
- New features (export, collections, weekly digest)

## How to contribute
1. Fork the repo
2. Create a branch
3. Make changes
4. Submit a PR

## Principles
- Keep it minimal
- Keep it readable
- Retrieval-first (search and sources matter most)
